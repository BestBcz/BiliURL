| 前缀          | 含义        | 会归入哪个分组            |
| ----------- | --------- | ------------------ |
| `feat:`     | 新特性功能     | ✅ `New`            |
| `fix:`      | 修复 bug    | 🛠 `Fixed`         |
| `refactor:` | 代码重构      | 🧽 `Refactor`      |
| `docs:`     | 文档更新      | 📚 `Documentation` |
| `style:`    | 样式优化（无逻辑） | 🎨 `Improved`      |
| `perf:`     | 性能优化      | 🎨 `Improved`      |
| `test:`     | 增加测试      | 🔄 `Other Changes` |
| `chore:`    | 杂项任务      | 🔄 `Other Changes` |
| `ci:`       | CI配置相关    | 🔄 `Other Changes` |
